### Post Writing Format

**Filename format:** `YYYY-MM-DD-lorem-ipsum-is-here.md`

```
---
layout: post
title: Lorem Ipsum is here!
description: A short brief about what this post is about.
keywords: keyword1, next keyword2, just keywords
---
```
